🔹 2. How to create a wallet, buy USDT and exchange for URSAL?

Step 1 – Create your wallet
  • • Download and install MetaMask (browser or mobile).
  • • Configure the Polygon (Matic) network within MetaMask.
Step 2 – Buy USDT on Polygon network
  • • Create an account on a reliable exchange (e.g.: Binance).
  • • Buy USDT and withdraw directly on the Polygon (MATIC) network to your MetaMask.

⚠️ Attention: make sure to select Polygon (MATIC) network when withdrawing USDT from Binance.

Step 3 – Connect to QuickSwap
  • • Access the official pool link.
  • • Connect your MetaMask to the platform.
Step 4 – Make the exchange
  • • Select USDT → URSAL.
  • • Confirm the transaction in MetaMask.
  • • Done! Now you own URSAL tokens in your wallet.
Step 5 – View your tokens
  • • If it doesn't appear automatically, manually add the URSAL token contract to MetaMask.
  • • The URSAL token will be visible in your balance.
